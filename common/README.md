# Common

#### Check for previous command status code
```sh
echo $?
```
- 0  : Success
- 1 : Failure

#### Run previous command again
```sh
!!
```
#### Create a file
```sh
touch -m ${file}
```
- -m : Change the DateTime of the file
#### Show all path variable
```sh
$PATH
```
#### Show content of file
```sh
cat ${file}
```
